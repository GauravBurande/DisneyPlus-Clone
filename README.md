You can visit the webapp here: [Disney Plus Clone](https://disneyplus-clone-642bb.web.app/)

My preffered weapons of choice to build this webapp:

* NodeJS
* ReactJS
* Styled Components
* Firebase for auth and databse
* Redux for state management
